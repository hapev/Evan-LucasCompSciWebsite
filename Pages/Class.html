[THE CLASS]

"Computers themselves, and software yet to be developed, will revolutionize the way we learn." -Steve Jobs

Honors Computer Science Class meets A, C, E, and G days, located in Gilman - 4. the time the class meets varies from day to day. Materials needed include laptops or other electronic devices. A knowledge of Java programming is recommended.

The goals for the class are:

  • Learn more about HTML and CSS
  • Learn Python
  • Have fun working as a team
  • Learning how to manipulate data through editing
  • Learning more programming languages
  • Learn in depth about complex computer science subjects
